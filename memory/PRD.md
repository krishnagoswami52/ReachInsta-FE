# ReachInsta - Premium SaaS Landing Website

## Original Problem Statement
Build a premium modern SaaS landing website for "ReachInsta" - a competitor intelligence platform for Instagram creators. Website for Paddle verification + real customer conversions. Must look like a real funded startup, trustworthy, polished, production-ready.

## User Personas
- Instagram creators (individual and professional)
- Social media agencies
- Social media managers
- Growth-focused brands

## Core Requirements (Static)
- Clean premium SaaS UI (Stripe/Notion/Linear/Framer inspired)
- Dark mode default with light mode toggle
- Greyscale base with kinetic orange (#FF4500) accent (60-30-10 rule)
- 11 pages: Home, Features, Pricing, About, Contact, FAQ, Login, Signup, Terms, Privacy, Refund
- Animated dashboard mockup with Recharts
- Responsive mobile + desktop design
- Sticky navbar with glassmorphism
- Trust badges (Secure Payments, GDPR Ready, Cancel Anytime, Trusted by Creators)
- Paddle-ready pricing buttons with data-plan attributes
